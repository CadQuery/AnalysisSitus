Ad-hoc checks
check-vertices-ori Checks if the vertices of the part are distinguishable by their orientation flags. A valid edge should have one FORWARD vertex and one REVERSED vertex to properly denote its topological extremities. As a result of some poorly implemented modeling operators, this rule may appear to be broken.
check-contours Checks if the contours of the part faces are closed (have no gaps). The optional 'tolerance' argument is used to tune the precision of coincidence test. You can find more details in the corresponding chapter.
