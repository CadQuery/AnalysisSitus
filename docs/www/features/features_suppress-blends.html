Suppress blends

The blend suppression algorithm of Analysis Situs is based on the approach presented by [Venkataraman et al, 2002] and adopted for the OpenCascade boundary representation.

The corresponding Tcl commands are listed below:

  • recognize-blends: performs blends recognition for the entire model.
  • kill-blend: recognizes and suppresses a blend chain starting from the selected candidate face.
  • kill-blends-inc: performs fully automatic incremental recognition & suppression procedure.

Check out the sources for implementation of the blend recognition and suppression algorithm.

Test data

Our blends recognition and suppression algorithm is being tested on the specifically prepared data set. We made this data set publicly available together with the basic version of the algorithm.
