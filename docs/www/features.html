Inspection

This block contains description of the shape interrogation facilities available in Analysis Situs. In general, using the inspection tools, it is possible to extract the following information:
  • Topological connectivity in form of graphs.
  • Host geometry of edges and faces (types of curves and surfaces, their properties).
  • Local tolerances, locations and orientations associated with the boundary elements.
  • Attributed adjacency graph.
  • Topology graph.
  • Check overall validity.
  • Check geometric tolerances.
  • Check naked (open) edges.
  • Check non-manifold edges.
  • Check dihedral angles.
  • Curvature plots and combs.
  • Gaussian and mean curvature maps.
  • Check if face contour is closed.
  • Check face contour.
  • Check face normal vectors.
